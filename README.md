# Gemini Banana Veo Service

A Next.js quickstart and studio application for Google's **Veo 3.1** (Video) and **Nano Banana Pro** (Gemini 3 Pro Image) models.

## Features

*   **Veo 3.1 Preview**: Text-to-Video & Image-to-Video generation with advanced parameter controls (Resolution, Aspect Ratio, Duration).
*   **Nano Banana Pro**: Unified Text-to-Image and Image-to-Image editing/composition.
*   **Modern UI**: Unified context bar for image management, chat-based history, and intuitive controls.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server (requires `GEMINI_API_KEY`):
    ```bash
    export GEMINI_API_KEY=your_key
    npm run dev
    ```
